# Memory

Iris Memory allows Iris to remember your learning preferences and past interactions across conversations, providing more personalized support over time. This feature is powered by [Memiris](/docs/overview/ecosystem), a dedicated memory service within the EduTelligence ecosystem.

## What Iris Remembers

When Memory is enabled, Iris stores summarized information about your interactions — for example:

- Topics you have asked about frequently
- Concepts you have struggled with
- Your preferred explanation style (examples vs. formal definitions, step-by-step vs. overview)
- Context from previous conversations that may be relevant to your current question

This means Iris can avoid repeating explanations you have already understood and focus on areas where you need more help.

:::info
Memory is currently active in **Course Chat** and **Lecture Chat**. Exercise-specific chats (programming and text exercises) do not create or use memories at this time.
:::

:::info
Iris does **not** store your personal identity, passwords, or unrelated personal information. Memory is limited to learning-relevant interaction summaries.
:::

## How to Enable or Disable Memory

You can toggle Memory during the [AI experience selection](./getting-started) when you first set up Iris. You can also enable or disable it at any time through your **learner profile settings** in Artemis.

- **Enabling** Memory starts personalization from that point forward.
- **Disabling** Memory stops new memories from being created. Existing memories are retained until you explicitly delete them.

## The Brain Icon

When Memory is active, you may see a small **brain icon** on messages in the chat. This icon indicates:

- How many existing memories were **used** to inform that particular response
- How many new memories were **created** from that interaction

This gives you transparency into when and how personalization is being applied.

## Managing Your Memories

You can view and manage all stored memories in your **learner profile settings** in Artemis:

- **View** what Iris has remembered about your learning interactions
- **Delete individual memories** that you do not want Iris to retain
- **Delete all memory data** to completely reset Iris's personalization for your account

:::warning
Deleting memories is permanent and cannot be undone. After deletion, Iris will no longer have the context from those previous interactions.
:::
